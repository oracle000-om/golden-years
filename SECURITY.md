# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Golden Years Club, please report it responsibly.

**Email:** [enter@daye.town](mailto:enter@daye.town)

Please include:

- A description of the vulnerability
- Steps to reproduce
- Any potential impact

We will acknowledge your report within 48 hours and work with you to resolve the issue before any public disclosure.

**Please do not open a public GitHub issue for security vulnerabilities.**
